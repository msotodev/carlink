# 🚗 Car Link

Aplicación web desarrollada con React + Vite para crear páginas rápidas y atractivas donde puedes mostrar un vehículo en venta (autos o motos) y compartirlo fácilmente mediante un código QR.

### ✨ Características
 - 📸 Galería de imágenes del vehículo
 - 📋 Detalles técnicos (marca, modelo, año, precio, etc.)
 - 📱 Diseño responsive (móvil y desktop)
 - 🔗 URL única para compartir o generar QR
 - ⚡ Carga rápida gracias a Vite
 - 🎨 Estilos modernos (Tailwind CSS recomendado)
